# Nazarii Kozak — сайт-портфоліо filmmaker'а

Простий статичний сайт (HTML/CSS/JS, без збірки) — деплоїться на Vercel за пару хвилин.

## Структура

```
index.html      — розмітка всіх секцій
styles.css      — стилі
script.js       — годинник Варшави в шапці, бургер-меню, drag-scroll реела, scroll-reveal
assets/
  hero-poster.jpg    — статичний кадр шапки (головне фото)
  hero-portrait.jpg  — фото в секції "Про мене"
  showreel.mp4       — не використовується зараз (шапка статична), лишив про запас
```

## Що обов'язково поміняти

1. **Соцмережі** — зараз посилання-заглушки (`href="#"`) в шапці й у секції "Контакт". Онови на реальні профілі Instagram/YouTube/Telegram.
2. **Email/телефон** — `hello@nazariikozak.com` і поле в лівому куті шапки — постав реальні контакти.
3. **Кадри в реелі** — зараз там кольорові плейсхолдери (`.reel__placeholder`). Заміни `<div class="reel__placeholder">` на `<video>` зі своїм файлом або `<iframe>` з YouTube/Vimeo.
4. **"Про мене"** — текст-заглушка, онови коли будеш готовий.
5. **Пакетні пропозиції** — блок у "Послугах" зараз загальний ("деталі за запитом"). Коли скинеш конкретні пакети — розпишу окремими картками.

## Логіка послуг

Ціни свідомо не вказані — кожен рядок веде на кнопку **"Подати заявку"** (`href="#contact"`), яка скролить до контактної форми/email. Це стосується й оператора (погодинна оплата) — ти просив ніде не показувати конкретні ставки.

## Деплой на Vercel

Найпростіше — через сайт:

1. Зайди на https://vercel.com → New Project.
2. Обери "Deploy without Git" / перетягни цю папку (drag & drop) — Vercel сам розпізнає статичний сайт.
3. Deploy — готово, отримаєш посилання на кшталт `your-project.vercel.app`.

Або через термінал (якщо є Node.js):

```bash
npm i -g vercel
cd videograf-site
vercel
```

Слідуй підказкам у консолі — на питання про framework обери "Other" (це звичайний статичний сайт).

## Нотатки по дизайну

- Тема: темний кінематографічний мінімалізм, акцент — ембер/помаранчевий відблиск (як на твоєму референсному фото).
- Шрифти: Bebas Neue (заголовки, кіно-афішний стиль) + Archivo (текст/UI).
- "Таймкод" у навбарі та нумерація секцій (00:01, 00:02...) — відсилання до монтажного таймлайну, а не декоративні цифри.
- Плівкове зерно (`.grain`) — легкий шум поверх усього сайту для кіно-текстури.
