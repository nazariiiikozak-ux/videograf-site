# Сайт-портфоліо відеографа

Простий статичний сайт (HTML/CSS/JS, без збірки) — деплоїться на Vercel за пару хвилин.

## Структура

```
index.html      — розмітка всіх секцій
styles.css      — стилі
script.js       — таймкод у шапці, бургер-меню, drag-scroll реела, scroll-reveal
assets/
  hero-portrait.jpg  — твоє фото (референс), зараз використовується як poster для відео і в секції "Про мене"
```

## Що обов'язково поміняти перед публікацією

1. **Ім'я / бренд** — заміни `ТВОЄ ІМ'Я` в `index.html` (тег `<title>`, `.hero__title`, футер) і `hello@tvoyeimya.com` на свій email.
2. **Відео в шапці** — постав файл, напр. `assets/showreel.mp4`, і онови `src` у `<source>` всередині `.hero__video`. Файл краще стиснути (H.264, до ~15–20 МБ, коротка петля 10–20 сек) — інакше сайт вантажитиметься повільно.
3. **Кадри в реелі** — зараз там кольорові плейсхолдери (`.reel__placeholder`). Заміни `<div class="reel__placeholder">` на `<video>` зі своїм файлом або `<iframe>` з YouTube/Vimeo.
4. **Тексти проєктів, послуг, цін, соцмереж** — заповни реальними даними.

## Деплой на Vercel

Найпростіше — через сайт:

1. Зайди на https://vercel.com → New Project.
2. Обери "Deploy without Git" / перетягни цю папку (drag & drop) — Vercel сам розпізнає статичний сайт.
3. Deploy — готово, отримаєш посилання на кшталт `your-project.vercel.app`.

Або через термінал (якщо є Node.js):

```bash
npm i -g vercel
cd videograf-site
vercel
```

Слідуй підказкам у консолі — на питання про framework обери "Other" (це звичайний статичний сайт).

## Нотатки по дизайну

- Тема: темний кінематографічний мінімалізм, акцент — ембер/помаранчевий відблиск (як на твоєму референсному фото).
- Шрифти: Bebas Neue (заголовки, кіно-афішний стиль) + Archivo (текст/UI).
- "Таймкод" у навбарі та нумерація секцій (00:01, 00:02...) — відсилання до монтажного таймлайну, а не декоративні цифри.
- Плівкове зерно (`.grain`) — легкий шум поверх усього сайту для кіно-текстури.
